Universidad

Tecnologias Dominadas

HTML

HiperText Markup Language, el lenguaje que maqueta el internet, hace varios años que conozco este lenguaje y, aunque no es un lenguaje de programacion, sabemos lo esencial que es para el internet desde sus inicios.

Dominio de esta tecnologia: 90/100.

CSS3

No podemos hablar de HTML sin tomar en cuenta a CSS, que es el lenguaje que la color y embellece la web, aprendí CSS casi en simultaneo con HTML ya que van muy de la mano, sobre todo, en el inicio como desarrollador web.

Dominio de esta tecnologia: 70/100.

JavaScript

JavaScript es la vida de la web, si HTML y CSS son el esqueleto, podemos decir que JS es el musculo que le da vida a la web para que podamos interactuar con ella.

Dominio de esta tecnologia: 60/100.

Git

Este tecnologia pasó de ser un complemento para organizar, a ser una necesidad para mantener y escalar proyectos, desde los más simples hasta los más complejos utilizan Git en su dia a dia y por eso es tan importante.

Dominio de esta tecnologia: 80/100.
